### 2. WAL 크기 장중 모니터링
**시점**: 09:30, 12:00, 15:00, 15:30

**모니터링 명령어**:
```bash
# WAL 크기 확인
ls -lh /root/webapp/backend/kisautotrade.db-wal

# 3MB 이상 시 체크포인트 실행
cd /root/webapp/backend
python3 << 'EOF'
import sqlite3
import os
from datetime import datetime

DB_PATH = "kisautotrade.db"
wal_path = DB_PATH + "-wal"
wal_size = os.path.getsize(wal_path) if os.path.exists(wal_path) else 0
print(f"WAL 크기: {wal_size / 1024 / 1024:.2f} MB ({datetime.now().strftime('%H:%M')})")

if wal_size > 10 * 1024 * 1024:  # 10MB 이상
    conn = sqlite3.connect(DB_PATH)
    conn.execute("PRAGMA wal_checkpoint(TRUNCATE)")
    conn.close()
    print("✅ 체크포인트 실행 완료")
EOF
```

**예상**:
- 09:30: ~5MB (1분봉 수집 진행 중)
- 12:00: ~8MB (오전 거래 완료)
- 15:30: ~10MB (하루 거래 완료)
