# RESEARCH-001: AI 비용 최적화 연구 보고 (7개 전략)

## 개요
AADS 시스템에서 사용하는 AI API 비용을 최적화하기 위한 7가지 전략 연구.

## 전략 1: 모델 티어 분리
- 단순 작업: claude-haiku-4-5 (저렴)
- 중간 복잡도: claude-sonnet-4-6 (중간)
- 복잡한 분석: claude-opus-4-6 (고가)
- 예상 절감: 40-60%

## 전략 2: 컨텍스트 압축
- 대화 이력 요약 후 재주입
- 불필요한 시스템 프롬프트 최소화
- 예상 절감: 20-30%

## 전략 3: 캐싱 레이어
- 동일 질의 결과 Redis 캐싱 (TTL: 1시간)
- 정적 콘텐츠 스냅샷 활용
- 예상 절감: 15-25%

## 전략 4: 배치 처리
- 실시간 불필요한 요청 큐잉
- 야간 배치 실행
- 예상 절감: 10-20%

## 전략 5: 프롬프트 최적화
- Few-shot 예시 최소화
- Chain-of-thought 필요 시만 사용
- 예상 절감: 10-15%

## 전략 6: 토큰 모니터링
- 월별 예산 알림 (80% 초과 시 Telegram 알림)
- 프로젝트별 비용 추적
- 예상 효과: 과잉 지출 예방

## 전략 7: 오픈소스 대체
- 단순 분류: 로컬 모델 (Ollama)
- 임베딩: text-embedding-3-small
- 예상 절감: 30-50% (특정 워크로드)

## 종합 절감 예상
현재 월 $200-400 → 최적화 후 $80-160 (약 55% 절감)
