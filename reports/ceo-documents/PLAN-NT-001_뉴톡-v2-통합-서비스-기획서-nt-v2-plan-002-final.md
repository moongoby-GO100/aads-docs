## 1. 프로젝트 개요

### 1.1 배경
뉴톡 V1(CodeIgniter 2.x / PHP 5.4)을 Laravel 12 + Next.js 16으로 전면 재구축.
단순 B2B 콘텐츠 중개 플랫폼에서 **SNS형 B2B SaaS 마켓플레이스**로 진화.

### 1.2 비전
"동대문 패션의 콘텐츠·거래·마케팅을 하나로 연결하는 올인원 커머스 SaaS"

### 1.3 핵심 전환
- V1: 도매 콘텐츠 제작 → 소매가 다운로드 → 자기 쇼핑몰에 등록 (거래 흐름 밖)
- V2: 도매 콘텐츠 생산(AI) → SNS 피드 발견 → 뉴톡 내 직접 사입 → 전채널 자동 등록 → 자동 마케팅 (거래 흐름 안)

## 4. 서비스 아키텍처 – 8대 레이어

### 레이어 1: SNS 소셜 엔진
- 홈 피드 (팔로우+AI 추천 혼합, 세로 스크롤)
- 익스플로어 (카테고리/스타일/트렌드 브라우징)
- 쇼츠 (15-60초 세로 영상, 풀스크린 스와이프)
- 스토리 (24시간 한정)
- DM (소매-도매 1:1)
- 팔로우/찜 시스템

### 레이어 2: 도매 SaaS 브랜드 인프라
- 브랜드 페이지 ({brand}.newtalk.kr)
- 거래처 제도 (소매→도매 신청/승인/전용가격)
- 주문·배송 관리 (사방넷 연동)
- 매출 분석 대시보드
- 소매 유치 도구 (QR, 추천 알고리즘)

### 레이어 3: 소매 커머스 허브
- 뉴톡 내 직접 사입 주문
- 내 쇼핑몰 원클릭 등록 (카페24)
- 위탁배송 (검품·포장·택배)
- 대량 등록 파일
- 교환·반품 대행

### 레이어 4: 마켓플레이스 거래 엔진
- 경로 A: 오픈 마켓플레이스 (승인 없이 주문)
- 경로 B: 브랜드 직거래 (거래처 승인 후 전용가)
- 통합 장바구니·결제
- 정산 시스템 (A: 3-5%, B: 1-2%)

### 레이어 5: AI 인텔리전스
- 트렌드 분석
- AI 사입 추천
- 소매 맞춤 추천 (브랜드/상품)

### 레이어 6: 확장 (일본·라이브)
- 일본 크로스보더 (번역/배송)
- 라이브 B2B (실시간 주문)

### 레이어 7: 셀러 확장 엔진
- 다채널 자동 등록 (카페24→네이버/쿠팡/11번가)
- 유튜브 쇼핑 연동 (쇼츠 자동 업로드+태그)
- SNS 연동 (인스타/틱톡/페이스북 자동 게시)
- 자동 마케팅 엔진 (시간최적화/해시태그/A·B테스트)
- 성과 분석 통합 대시보드

### 레이어 8: 콘텐츠 팩토리
- Phase 1: 뉴톡 스튜디오 (사람+시스템)
  입고→MD분류→촬영→NAS→AI보조편집→상세페이지→QA→게시
- Phase 2: AI 생성 확장
  배경제거/가상모델/코디조합/쇼츠생성/상세페이지 카피
- 등급: 베이직(AI만)/프로(AI+스튜디오)/프리미엄(전담MD)
