# AADS Documentation

AI Autonomous Development System - 공식 문서 저장소

## 프로젝트 개요
아이디어 입력 → AI 자동 기획 → 설계 → 개발 → 테스트 → 배포 → 운영 → 고도화

## 문서 구조
- `/reports` - 기획 보고서, 비용 분석, 시장 조사
- `/architecture` - 시스템 아키텍처, 기술 문서
- `/meeting-notes` - 의사결정 기록
- `/phase-reports` - 단계별 진행 보고서
- `/specs` - 상세 기술 사양서

## 타임라인
| Sprint | 기간 | 목표 | 상태 |
|--------|------|------|------|
| Sprint 1 | Week 1-2 | 인프라, SDK, DB | 🔄 진행중 |
| Sprint 2 | Week 3-4 | 에이전트, 게이트, 대시보드 | ⏳ 대기 |
| Sprint 3 | Week 5-6 | CI/CD, 배포, 라우팅 | ⏳ 대기 |
